## 결론

## 공개 페이지 계약

목록은 `GET https://www.swcf.or.kr/?p=30&curYear=YYYY&curMonth=M`으로 월을 이동한다. 공개 표의 각 행은 다음 사실을 제공한다.

- 분류: 어린이·가족·전체·성인 등
- 교육기간
- 접수 상태: 접수중·접수대기·마감
- 제목과 숫자형 `idx`
- 장소

공식 상세 `GET https://www.swcf.or.kr/?p=30_view&idx=<숫자>`는 교육분류, 접수기간·방법, 교육기간·시간, 교육장소, 이용료, 주최·주관, 문의처를 공개한다. 본문에 `참여대상`, `교육대상`, `운영대상`처럼 명시된 대상 문장이 있으면 한 줄만 추출해 연령 단서로 사용한다. 본문 전체·이미지·신청 링크는 복제하지 않는다.

robots는 `/cmsMng/`와 검색 파라미터 경로를 금지하지만 `/?p=30`, `/?p=30_view&idx=<숫자>`는 금지하지 않는다. 크롤러는 실행 때마다 robots를 다시 판정하며, 정책을 읽지 못하거나 HTML/WAF가 돌아오면 실패 닫힘 처리한다.

## 필터와 중복 방지

전용 source ID는 `suwon_culture_foundation_education`이다.

1. `CrawlWindow`가 걸치는 모든 달을 순서대로 조회하되 24개월을 상한으로 둔다.
2. 공식 호스트의 `p=30_view`와 숫자형 `idx`만 허용하고 나머지 쿼리는 canonical URL에서 제거한다.
3. 같은 `idx`가 여러 달 목록에 반복돼도 상세는 한 번만 읽고 이벤트도 한 번만 만든다.
4. 어린이·가족·유아·초등 단서는 우선 포함한다.
5. `전체` 분류라도 체험·방학·공예·놀이·전통·생태·과학·미디어 등 가능성이 있으면 낮은 관련도 후보로 보존한다.
6. 목록 분류가 성인 전용이면 제외한다. 상세 대상이 성인만으로 확인된 경우도 제외한다.
7. 상세의 실제 교육기간이 요청 창과 겹치는 이벤트만 반환한다.

마감 프로그램도 사실 상태를 그대로 수집한다. 사용자 조회 API의 기본값은 마감 이벤트를 숨기므로, 과거 발견 이력과 상태 변화는 보존하면서 일반 알림에는 노출되지 않는다.
