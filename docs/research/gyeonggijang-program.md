## 결론

- 2026 운영기간: 2026-04-18 ~ 2026-12-15

- 운영 협력: 경기관광공사, 경기문화재단, 한국도자재단, 경기콘텐츠진흥원, 경기아트센터, DMZ국제다큐영화제

## 공식 공개 목록 계약

경기문화재단 메인 페이지의 1차 JavaScript가 아래 공개 JSON을 직접 호출한다.

- `GET https://www.ggcf.kr/api/events?progress=soon&limit=100&page=1`
- `GET https://www.ggcf.kr/api/edus?progress=soon&limit=100&page=1`

`progress=soon`은 공식 메인 페이지가 진행 중·진행 예정 콘텐츠를 묶어 보여줄 때 사용하는 값이다. 두 분류를 모두 페이지네이션해야 행사로 올라온 체험과 교육으로 올라온 체험을 함께 잡을 수 있다. 응답의 `last_page`까지 읽되 서비스의 `CrawlWindow.max_pages` 상한을 지킨다.

전용 필터는 `title + summary + place + affiliationName`을 정규화한 뒤 `컬처라운지경기장`이 들어간 행만 통과시킨다. 이 방식은 다음 두 경우를 모두 잡는다.

1. 제목에 `[컬처라운지 경기,장]`이 붙은 프로그램
2. 현재 프로그램처럼 제목에는 공간명이 없지만 `place`가 `경기도 컬처라운지 경기, 장`인 프로그램

반대로 `수원월드컵경기장`, `종합경기장` 등 일반 경기장은 `컬처라운지` 표지가 없으므로 제외된다. `display=none`, 비공식 호스트, 숫자형 공식 상세 경로가 아닌 링크도 제외한다.

공식 robots 파일은 첨부·업로드 경로만 금지하고 행사 API와 정보 상세 경로는 금지하지 않는다. [경기문화재단 robots.txt](https://www.ggcf.kr/robots.txt)

### 공식 데이터 내부 불일치 처리

현재 공식 JSON/상단 구조화 필드는 시작일을 `2026-07-14`로 표시하지만 같은 공식 상세 본문의 `운영일정`은 `2026-07-12`와 `2026-07-26` 세션을 명시한다. 크롤러는 다음처럼 양쪽을 잃지 않는다.

- `Event.event_start/event_end`: 상세 본문의 명시적 운영 세션 첫날·마지막 날 사용
- `raw.event_period`: JSON 구조화 값 `2026-07-14(화) ~ 2026-07-26(일)` 보존
- `raw.published_schedule`: 상세 본문의 운영일정 문장 보존

이렇게 하면 공식 원문의 충돌을 숨기지 않으면서 실제 공개 세션을 알림 범위에 포함한다.

## 수집 경계

호출하는 경로는 다음뿐이다.

1. 공개 `events` JSON 목록
2. 공개 `edus` JSON 목록
3. 통과한 행의 `https://*.ggcf.kr/events/<숫자>` 또는 `/edus/<숫자>` 정보 상세

다음 경로는 링크가 상세 HTML에 있어도 추출·호출하지 않는다.

- 네이버 예약
- `접수하기` 동작
- 로그인·회원가입
- 본인인증
- 캡차·대기열
- 예약 가능 좌석 조회
- 신청·결제·예약 제출
- robots에서 금지한 첨부·업로드 이미지 경로

공식 인스타그램은 중요한 선행 발견 채널이므로 출처 목록에는 유지한다. 다만 소셜 플랫폼의 로그인·전달 제약을 우회하는 크롤러는 만들지 않고, 발견된 프로그램은 경기문화재단 공개 목록·상세로 확인된 뒤 정규 데이터로 저장한다.
